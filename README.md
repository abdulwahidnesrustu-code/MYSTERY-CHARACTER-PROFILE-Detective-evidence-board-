# Detective Evidence Board -Mystery Character Profile

This project is a single-page detective-style webpage that presents the investigation of an unknown subject using an evidence board layout. The page is designed using basic HTML and inline CSS.

## Overview
The webpage organizes case information into structured sections including the subject profile, physical description, confirmed facts, timeline, collected evidence, psychological analysis, detective theories, and information links.

## Technologies Used
HTML  
Inline CSS  

## Features
Organized evidence board layout  
Simple and readable structure  
Single page design  

## How to Run
Download or clone the repository and open the `WEB DESIGN PROJECT_ABDULWAHID_NESRU.html` file in a web browser.
